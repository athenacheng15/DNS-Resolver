COMP3331/9331 DNS resolver assignment resources
Release: 2026T2-v1

This package contains small sample inputs for local development.
They are not a full test suite. Do not hard-code them.

Contents:

named.root
    Sample root hints file for resolver testing.

*.bin
    Binary DNS messages for Stage 1 parser testing.
    Query files contain DNS queries.
    Response files contain DNS responses with sample resource records.

CHECKSUMS.txt
    SHA-256 checksums for the files in this package.

Use the assignment specification for the required behaviour.
